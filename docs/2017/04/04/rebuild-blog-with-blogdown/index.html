blogdownでブログを作成したメモ

Published Tue, Apr 4, 2017 by Kazuki Tamura

blogdownを使ってblogを再構築してみました。日本語の情報はそれほど多くないです。

  • Rとblogdownでかんたんにgithub.io上にブログを使ってみよう！！
  • blogdownのTips

以下は自分用の備忘録です。ただし，すでにうろ覚えですが。

  1. homebrewでhugoをインストール
  2. githubにリポジトリblogを作成
  3. blogをローカルにgit clone
  4. README.mdを削除し，git push
  5. RStudioでsetwd("blog")しておいてblogdown::new_site()する（この時点ではblogフォルダは空っぽじゃないといけないので，まだRStudioのプロジェクトを作成できない）
  6. 必要に応じてテーマをインストールblogdown::install_theme("jbub/ghostwriter")

  7. config.tomlに以下を以下を追加

    publishDir = "docs"
hasCJKLanguage = true
languageCode = "ja-jp"
  8. blogにRStudioのプロジェクトを作成
  9. 諸々の設定ファイルを修正，不要なファイルを削除
  10. blogdown::serve_site()でプレビューしながら記事を作成
  11. blogdown::build_site()を実行
  12. RStudioの[Git]タブからgit pushする

  13. githubのblogリポジトリのSettingsで，Github PagesのSourceをmaster branch /docs folderに変更する
